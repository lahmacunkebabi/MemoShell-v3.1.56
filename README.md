# MemoShell v3.1.69

Developer: Memo's Projects

## Özellikler

1. **Dosya İndir ve Klasöre Çıkar** - Link'ten dosya indir ve arşivden çıkar
2. **Dosya veya Klasör Sil** - İstediğin yolu sil
3. **Dosya Hash (Güvenlik) Kontrolu** - SHA256 hash'ini kontrol et
4. **Çıkış** - Programı kapat

## Klasör Yapısı

```
MemoShell/
├── MemoShell.bat (Ana dosya)
├── README.md
└── modules/
    ├── indir.bat
    ├── sil.bat
    └── hash.bat
```

## Kullanım

Ana dosya olan `MemoShell.bat` dosyasını çalıştır ve menüden istediğin seçeneği sec.
